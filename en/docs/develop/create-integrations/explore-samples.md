# Explore sample integrations

WSO2 Integrator includes a gallery of built-in sample projects that cover common integration scenarios. Use them to learn integration patterns or as a starting point for your own projects.

## Browsing samples

In the WSO2 Integrator sidebar, click **Explore**. The sample gallery opens, showing projects organized by category:

- **Automations** — Scheduled and trigger-based automation examples
- **Integration as API** — HTTP, GraphQL, gRPC, and WebSocket service examples
- **File Integration** — HTTP, GraphQL, gRPC, and WebSocket service examples
- **AI Agent** — Agent and LLM integration patterns

Browse by category or use the search bar to find a sample by name or keyword.

## Opening a sample

Select a sample from the gallery to view its description. Choose a local directory where the project will be created, then click to open it. WSO2 Integrator downloads the sample and creates the project at the selected path.

Once the project loads, it opens in the appropriate view based on the project type:

- A service or automation opens in the [Integration View](../project-views/integration-view.md)
- A workspace project opens in the [Workspace View](../project-views/workspace-view.md)
- A shared library opens in the [Library View](../project-views/library-view.md)

From there you can run, debug, or try the integration using the built-in tools.
