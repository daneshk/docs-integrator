# WSO2 Integrator

**Build integrations with low-code simplicity and pro-code power.**

WSO2 Integrator is a Ballerina-powered integration platform that allows you to connect anything and solve any integration challenge. It combines low-code simplicity with pro-code power, enabling you to build AI agents, APIs, data, and events across cloud, on-prem, and hybrid environments.

## Integration Areas

### 1. Get Started
Install, set up, and build your first integration in under 10 minutes.
- **[Connectors Catalog](/docs/connectors)**: Browse 200+ pre-built connectors.
- **[Quick Start: REST API](/docs/get-started/quick-start-api)**: Build your first API integration.
- **[Build an AI Agent](/docs/get-started/quick-start-ai-agent)**: Get started with AI-driven integration.

### 2. Develop
Build services, transform data, and test integrations on your machine.
- **Integration Artifacts**: Create services, listeners, and endpoints.
- **Data Mapper**: Visual data mapping and transformation.
- **Unit Testing**: Test your integrations with a built-in framework.

### 3. Connectors
Browse 200+ pre-built connectors for SaaS (Salesforce, ServiceNow), databases (MySQL, MongoDB), messaging (Kafka, RabbitMQ), and AI (OpenAI, Anthropic).

### 4. GenAI
Build AI-powered integrations with agents, RAG, and MCP servers. Leverage LLMs directly within your integration workflows.

### 5. Tutorials
End-to-end walkthroughs, integration patterns, and sample projects. Learn how to solve real-world integration problems.

### 6. Deploy & Operate
Docker, Kubernetes, CI/CD, observability, and production security.
- **Docker & Kubernetes**: Deploy with first-class support for cloud-native containers.
- **CI/CD**: Automate your development lifecycle with GitHub Actions or Jenkins.
- **Observability**: Monitor and debug with ICP and built-in tracing.

### 7. Reference
Language reference, configuration keys, CLI commands, and error codes. Comprehensive documentation for professional developers.
